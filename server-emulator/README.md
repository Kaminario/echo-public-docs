# Sample Server

The server implements an API for simplifying `Silk Instant Extract` instrumentation.

It provides an example responses for an APIs required to integrate `Silk Instant Extract` functionality in to your code.

## Directory content

- setup-env.sh

  The environment setup to run the server

- start-server.sh
  used to start server

  The Server accepts following arguments:

  |parameter|description|
  |-|-|
  |--host|The host name of the server (Default: 0.0.0.0)|
  |--port|The port number of the server (Default: 8000)|

- start-client.sh

  start clent example (written in python)

- dump_openapi.sh

  Dumping openapi.json from the server.

requirements:

- linux based os

- python>=3.10 installed

## Install and Run server

We assume curent directory is:

`instant-extract-instrumentation/example`

### Install environment

The Server implemented using python, Uvicorn is an ASGI web server, and FastAPI web framework.

```bash
./setup-env.sh
```


### Run server

```bash
./start-server.sh
```

## Open API Spec

  The file `openapi.json` is auto generated by FastAPI using dump_openapi.sh

  It also awailable on running server at
  `http://localhost:8000/docs`
